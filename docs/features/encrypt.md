# 加密

:::tip

请知悉：v2.0.1及之后，管理员登陆之后无视加密。

:::

在后台元信息处添加记录，填写路径和密码即可，其中的路径：是访问alist页面时的pathname，如要设置`https://alist.nn.ci/本地存储`则路径是`/本地存储`，如：

- https://alist.nn.ci -> `/`
- https://alist.nn.ci/a -> `/a`
- https://alist.nn.ci/a/b -> `/a/b`

正确填写路径之后，填写密码字段即可。
